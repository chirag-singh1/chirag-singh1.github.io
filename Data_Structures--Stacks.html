Stacks


Data structure where insertion and deletion occur on the same end
Follows LIFO or FILO ordering
Main methods:
stack.push() adds item to top of stack
stack.pop() removes and returns top item from the stack
stack.peek() returns top element from the stack
stack.isEmpty() returns true if there are no elements in stack
Access: O(N), Search: O(N), Insertion: O(1), Deletion: O(1), Space: O(N)
Applications: string reversal, undo/redo features, graph traversal (depth-first)
