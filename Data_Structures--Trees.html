Trees


Hierarchical data structure in which each node links to more nodes
Top node called the root node, each node has a value and pointer to child nodes
Applications: Naturally store hierarchical data, especially binary search data
Maximum number of nodes per level: 2^l, maximum number of nodes is 2^h-1
Access/Search/Insertion/Deletion: O(log N), Space: O(N)
TreeNode class is not standardized, will generally use node.left and node.right to access
