Hashing


Any algorithm that can be used to map arbitrary size data to fixed-size values
Each data value is hashed and assigned a function as a result of the hashing function
Collisions where two data values have the same result can be handled by chaining them (each cell of hash table points to linked list that holds same hash value, requires memory) or open addressing (each value is stored in the hash table, where each cell is either a value or nothing, requires search time)
Use: store key/value pairs in hash table with O(1) access time
Implemented by classes like HashSet and HashMap (where hash key is memory address)
Commonly used for security applications
