Primitive Data Types


Can be compared with == operator
Casting:
Implicit: Magnitude of value is preserved, precision may be lost (conversion to more precise data type)
Explicit: Magnitude of numeric value not necessarily preserved (conversion to less precise data type)
Have fixed memory size
Wrapper classes store primitive data types as an object

byte: 8-bit signed 2’s complement integer (-128 to 127)
short: 16-bit signed 2’s complement integer (-32,768 to 32,767)
int: 32-bit signed 2’s complement integer (-2^31 to 2^31-1)
Contains Integer wrapper class: Integer.parseInt(string)
long: 64-bit 2’s complement integer (-2^63 to 2^63-1)
float: single-precision 32-bit floating point decimal
double: double-precision 64-bit floating point decimal
Contains Double wrapper class: Double.parseDouble(string)
boolean: true/false (1 bit of information)
Not represented as 0/1, cannot be casted to any other data type
char: 16-bit Unicode character
Contains Character wrapper class: Character.getNumericValue(char), Character.isDigit(char), Character.isLetterOrDigit(char), Character.isAlpha(char), Character.isUpperCase(char), Character.toLowerCase(char)
