HashMap


Stores data in pairs of keys and values
Each key can only correspond to one value
Important methods:
map.containsKey(key)returns true if key is in the keyset
map.containsValue(value) returns true if some key maps to value
map.get(key)returns the value corresponding to the key
map.entrySet() returns set view of key/value set
map.isEmpty() returns true if the map is empty
map.keySet() returns set view of keys
map.put(key,value) enters key, value pair to map
map.remove(key) removes key and corresponding value
map.size() returns size of map
