Priority Queues


Same as queue, but each element now has a priority
High priority elements dequeued before low priority elements (same priority served according to FIFO)
Most efficiently implemented using a linked list (do not have to delete items when deleting highest priority)
Elements must be sortable according to Comparator (increasing values)
Main methods all the same as queues
Insertion/Deletion: O(log N), Search/Access: O(1), Space: O(N)
