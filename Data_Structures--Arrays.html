Arrays


Fixed-size sequential collection of elements (will automatically allocate enough memory for the length of the array)
Passed by value to functions (can change contents of array, but not variable that points to it in caller)
Most useful in cases where single data points must be stored with a given length
Access: O(1), Search: O(N), Insertion: O(N), Deletion: O(N), Space: O(N)
