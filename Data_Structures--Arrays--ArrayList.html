ArrayList


Linear data structure in which elements are not stored in continuous locations
Each node contains a pointer to one or more other nodes
Can be used over array in cases where there will be many insertions and deletions and dynamic size is needed
Worse cache locality (decreases overall performance)
Access: O(N), Search: O(N), Insertion: O(1), Deletion: O(1), Space: O(N)
Main Methods: LinkedList class implements List interface
Singly linked list: Each node has value and pointer to next node
Doubly linked list: Each node has value and pointer to next node and previous node
