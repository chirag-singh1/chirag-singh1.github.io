String Methods


//Generally initialized to string literal
//Can also be initialized with 
new String() 
//to empty String or 
new String(char [])

String string;
string.charAt(index) //returns character
string.compareTo(string) //returns negative if alphabetically before other string
string.equals(string) //determines equality
string.indexOf(char) //returns index of first instance of character
string.indexOf(char, index) //returns index of first instance of character after index
string.length() //returns length of string
string.replaceAll(regex, replacement) //replaces all instances of regex
string.split(regex) //returns string array split over regex
string.substring(index1, index2) //returns string from index1 to index2-1
string.toUpperCase() //returns uppercase version of string
string.trim() //returns copy of string without leading/trailing whitespace
